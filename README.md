# 🚀 Featured Project

# 🏋️ Track Fit – AI Fitness Tracking Platform

## 🔥 Overview

Track Fit is a modern AI-powered fitness tracking platform designed with a premium UI/UX experience that helps users monitor activities, food intake, calories, and overall fitness progress.

The platform integrates **Gemini AI** for intelligent food analysis and smart health insights.

---

## ⚡ Features

✅ AI Food Analysis  
✅ Activity Tracking Dashboard  
✅ Authentication System  
✅ Dark & Light Mode  
✅ Responsive Modern UI  
✅ Analytics & Progress Tracking  
✅ MERN Stack Architecture  
✅ Smart Health Insights using Gemini AI

---

## 🧠 Tech Used

`React` `Node.js` `MongoDB` `Express.js` `Tailwind CSS` `TypeScript` `Gemini AI`
